character_species
/datum/character_species

since we're a server of snowflakes, we have this to embody character species

they store real species UID so we can change species, and allow us to tweak the species on spawn.

/datum/character_species is a singleton type stored on SScharacters. Intrinsics Defaults to set for visuals Defaults to set for background - typepaths allowed Languages - IDs only, as typepaths are too expensive to resolve
