Citadel Station 13 - Roleplay - Modules - Types — Var Details

modifier
/datum/modifier

This is a datum that tells the mob that something is affecting them. The advantage of using this datum verses just setting a variable on the mob directly, is that there is no risk of two different procs overwriting each other, or other weirdness. An excellent example is adjusting max health. Now for all the different effects. Percentage modifiers are expressed as a multipler. (e.g. +25% damage should be written as 1.25)

Vars

accuracyPositive numbers makes hitting things with guns easier, negatives make it harder.
accuracy_dispersionPositive numbers make gun firing cover a wider tile range, and therefore more inaccurate. Negatives help negate dispersion penalties.
attack_speed_percentMakes the holder's 'attack speed' (click delay) shorter or longer.
bleeding_rate_percentAdjusts amount of blood lost when bleeding.
client_colorIf set, the client will have the world be shown in this color, from their perspective.
descDitto.
disable_duration_percentAdjusts duration of 'disables' (stun, weaken, paralyze, confusion, sleep, halloss, etc) Setting to 0 will grant immunity.
evasionPositive numbers reduce the odds of being hit. Negative numbers increase the odds.
expire_atworld.time when holder's Life() will remove the datum. If null, it lasts forever or until it gets deleted by something else.
flagsFlags for the modifier, see mobs.dm defines for more details.
hasteIf set to 1, the mob will be 'hasted', which makes it ignore slowdown and go really fast.
hiddenIf true, it will not show up on the HUD in the Future(tm)
holderThe mob that this datum is affecting.
icon_scale_x_percentMakes the holder's icon get scaled wider or thinner.
icon_scale_y_percentMakes the holder's icon get scaled taller or shorter.
icon_stateSee above.
incoming_brute_damage_percentOnly affects bruteloss.
incoming_clone_damage_percentOnly affects cloneloss.
incoming_damage_percentAdjusts all incoming damage.
incoming_fire_damage_percentOnly affects fireloss.
incoming_hal_damage_percentOnly affects halloss.
incoming_healing_percentAdjusts amount of healing received.
incoming_oxy_damage_percentOnly affects oxyloss.
incoming_tox_damage_percentOnly affects toxloss.
light_colorIf set, the mob possessing the modifier will glow in this color. Not implemented yet.
light_intensityDitto. Not implemented yet.
light_rangeHow far the light for the above var goes. Not implemented yet.
max_health_flatAdjusts max health by a flat (e.g. +20) amount. Note this is added to base health.
max_health_percentAdjusts max health by a percentage (e.g. -30%).
metabolism_percentAdjusts the mob's metabolic rate, which affects reagent processing. Won't affect mobs without reagent processing.
mob_overlay_stateIcon_state for an overlay to apply to a (human) mob while this exists. This is actually implemented.
nameMostly used to organize, might show up on the UI in the Future(tm)
on_created_textText to show to holder upon being created.
on_expired_textText to show to holder when it expires.
originA weak reference to whatever caused the modifier to appear. THIS NEEDS TO BE A MOB/LIVING. It's a weakref to not interfere with qdel().
outgoing_melee_damage_percentAdjusts melee damage inflicted by holder by a percentage. Affects attacks by melee weapons and hand-to-hand.
pain_immunityMakes the holder not care about pain while this is on. Only really useful to human mobs.
pulse_modifierModifier for pulse, will be rounded on application, then added to the normal 'pulse' multiplier which ranges between 0 and 5 normally. Only applied if they're living.
pulse_set_levelPositive number. If this is non-null, it will hard-set the pulse level to this. Pulse ranges from 0 to 5 normally.
slowdownNegative numbers speed up, positive numbers slow down movement.
stacksIf true, attempts to add a second instance of this type will refresh expire_at instead.
vision_flagsVision flags to add to the mob. SEE_MOB, SEE_OBJ, etc.
wire_colors_replaceIf set, the client will have wires replaced by the given replacement list. For colorblindness.
