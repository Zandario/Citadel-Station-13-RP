soundbyte
/datum/soundbyte

managed sound file datums holding filename and specific metadatas

WARNING WARNING: If you use /grouped, you better stick with it, because probabilities for pick() get messed up if you have one grouped and one normal soundbyte (the normal will get it 50% of the time), because for Speed and Efficiency(tm) we are not going to distinguish between the two in the list pick()!

todo: autodetect length with rust

currently holds:

  • path/file - required for preload
  • name - defaults to filename
  • length - manual set for now, rust later tm
  • id - defaults to path for preloaded, otherwise should be made in init
