Intrinsics

Vars

abstract_typeAbstract type.
accessUseful for servers which either have fewer players, so each person needs to fill more than one role, or servers which like to give more access, so players can't hide forever in their super secure departments (I'm looking at you, chemistry!).
account_allowedDoes this job type come with a station account?
alt_titlesList of alternate titles; There is no need for an alt-title datum for the base job title.
assignableShould it show up on things like the ID computer?
current_positionsHow many players have this job.
department_accountsWhich department accounts should people with this position be given the pin for?
departmentsList of departments this job belongs to, if any. The first one on the list will be the 'primary' department.
departments_managedIs this a management position? If yes, list of departments managed. Otherwise null.
descDescription of the job
economic_modifierWith how much does this job modify the initial account amount?
factionFaction this job is considered part of, for the future considerations of "offmap"/offstation jobs
flagBitflags for the job.
has_headsetDo people with this job need to be given headsets and told how to use them? E.g. Cyborgs don't.
idID of the job, used for save/load
idtypeType of ID that the player will have. This is banned. Use outfits, this is only kept in for legacy.
join_typesDetermines if this job can be spawned into by players
minimal_accessUseful for servers which prefer to only have access given to the places a job absolutely needs (Larger server population).
minimal_player_ageIf you have use_age_restriction_for_jobs config option enabled and the database set up, this option will add a requirement for players to be at least minimal_player_age days old. (meaning they first signed in at least that many days before.)
mob_typeBitflags representing mob type this job spawns
offmap_spawnDo we require weird and special spawning and datacore handling?
outfit_typeWhat outfit datum does this job use in its default title?
req_admin_notifyIf this is set to 1, a text is printed to the player when jobs are assigned, telling him that he should let admins know that he has to disconnect.
selection_colorSelection screen color
sorting_orderUsed for sorting jobs so boss jobs go above regular ones, and their boss's boss is above that. Higher numbers = higher in sorting.
spawn_positionsHow many players can spawn in as this job.
supervisorsSupervisors, who this person answers to directly.
titleThe name of the job , used for preferences, bans and more. Make sure you know what you're doing before changing this.
total_positionsHow many players can be this job.
