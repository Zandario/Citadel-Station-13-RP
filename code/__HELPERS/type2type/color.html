code/__HELPERS/type2type/color.dm

/proc/rgb2hsl Convert RBG to HSL
/proc/hsl2rgb Convert HSL to RGB
/proc/hue2rgb Convert hue to RGB
/proc/heat2colour Convert Kelvin to RGB
/proc/heat2colour_r Convert Kelvin for the Red channel
/proc/heat2colour_g Convert Kelvin for the Green channel
/proc/heat2colour_b Convert Kelvin for the Blue channel
/proc/color_hex2num Assumes format #RRGGBB #rrggbb
/proc/color_hex2color_matrix ! Word of warning: Using a matrix like this as a color value will simplify it back to a string after being set.
/proc/color_matrix2color_hex Will drop all values not on the diagonal.
/proc/hex2rgb Converts a hexadecimal color (e.g. #FF0050) to a list of numbers for red, green, and blue (e.g. list(255,0,80) ).

The three procs below require that the '#' part of the hex be stripped, which hex2rgb() does automatically.
