/proc/get_atom_on_turf Returns the atom sitting on the turf. For example, using this on a disk, which is in a bag, on a mob, will return the mob because it's on the turf.
/proc/pick_mobless_turf_if_exists Picks a turf without a mob from the given list of turfs, if one exists. If no such turf exists, picks any random turf from the given list of turfs.
/proc/pick_random_edge_turf Picks a turf that is clearance tiles away from the map edge given by dir, on z-level Z.
