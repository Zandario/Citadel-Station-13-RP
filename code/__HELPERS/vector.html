/datum/plot_vectorplot_vector is a helper datum for plotting a path in a straight line towards a target turf. This datum converts from world space (turf.x and turf.y) to pixel space, which the datum keeps track of itself. This should work with any size turfs (i.e. 32x32, 64x64) as it references world.icon_size (note: not actually tested with anything other than 32x32 turfs).
/datum/vector_locvector_loc is a helper datum for returning precise location data from plot_vector. It includes the turf the object is in as well as the pixel offsets.
