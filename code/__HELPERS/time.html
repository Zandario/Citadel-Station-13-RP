code/__HELPERS/time.dm

/proc/time_stamp ISO 8601
/proc/station_time This is used for displaying the "station time" equivelent of a world.time value Calling it with no args will give you the current time, but you can specify a world.time-based value as an argument.
/proc/isDay Returns 1 if it is the selected month and day.
/var/next_duration_update Get the current month. Get the current day.
/var/midnight_rollovers 1/10 of a second, not real milliseconds but whatever. Not really needed, but I'll leave it here for refrence.. or something.
/proc/DisplayTimeText Get the current day. Takes a value of time in deciseconds. Returns a text value of that number in hours, minutes, or seconds.
