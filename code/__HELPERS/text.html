code/__HELPERS/text.dm

/proc/format_table_name Holds procs designed to help with filtering text Contains groups: ! SQL sanitization ! Text sanitization ! Text searches ! Text modification ! Misc
/proc/sanitize ! Text sanitization Used for preprocessing entered text.
/proc/sanitize_simple_tg Removes a few problematic characters. Renamed because namespace.
/proc/sanitizeSafe Run sanitize(), but remove <, >, " first to prevent displaying them as > < &34; in some places, after html_encode(). Best used for sanitize object names, window titles. If you have a problem with sanitize() in chat, when quotes and >, < are displayed as html entites - this is a problem of double-encode(when & becomes &), use sanitize() with encode=0, but not the sanitizeSafe()!
/proc/sanitizeName Filters out undesirable characters from names. Uppercase Characters: A .. Z Lowercase Characters: a .. z Number Characters: 0 .. 9 Symbol Characters: ' - . Hardcoded Characters: ~ | @ : # $ % & * + Space Character
/proc/reject_bad_text Returns null if there is any bad text in the string.
/proc/stripped_input Used to get a properly sanitized input, of max_length no_trim is self explanatory but it prevents the input from being trimed if you intend to parse newlines or whitespace.
/proc/stripped_multiline_input Used to get a properly sanitized multiline input, of max_length.
/proc/sanitize_old Old variant. Haven't dared to replace in some places.
/proc/dd_hasprefix ! Text searches
/proc/dd_hasprefix_case Checks the beginning of a string for a specified sub-string. This proc is case sensitive. Returns the position of the substring or 0 if it was not found.
/proc/dd_hassuffix Checks the end of a string for a specified substring. Returns the position of the substring or 0 if it was not found.
/proc/dd_hassuffix_case Checks the end of a string for a specified substring. This proc is case sensitive. Returns the position of the substring or 0 if it was not found.
/proc/replace_characters ! Text modification
/proc/add_zero Adds 'u' number of zeros ahead of the text 't'.
/proc/add_lspace Adds 'u' number of spaces ahead of the text 't'.
/proc/add_tspace Adds 'u' number of spaces behind the text 't'.
/proc/trim_left Returns a string with reserved characters and spaces before the first letter removed.
/proc/trim_right Returns a string with reserved characters and spaces after the last letter removed.
/proc/trim Returns a string with reserved characters and spaces before the first word and after the last word removed.
/proc/capitalize Returns a string with the first element of the string capitalized.
/proc/autocorrect Syntax is "stringtoreplace"="stringtoreplacewith".
/proc/strip_html_properly This proc strips html properly, remove < > and all text between for complete text sanitizing should be used sanitize()
/proc/stringmerge This proc fills in all spaces with the "replace" var (* by default) with whatever is in the other string at the same spot (assuming it is not a replace char). This is used for fingerprints.
/proc/stringpercent This proc returns the number of chars of the string that is the character. This is used for detective work to determine fingerprint completion.
/proc/TextPreview Used in preferences' SetFlavorText and human's set_flavor verb Previews a string of len or less length
/proc/copytext_preserve_html Alternative copytext() for encoded text, doesn't break html entities (" and other)
/proc/contains_az09 Uppercase Characters: A .. Z Lowercase Characters: a .. z Number Characters: 0 .. 9
/proc/strip_improper Strip out the special beyond characters for \proper and \improper from text that will be sent to the browser.
/proc/GenerateKey Random password generator. // Could've just flustered a bottom.
/proc/apply_text_macros Used for applying byonds text macros to strings that are loaded at runtime.
/proc/readd_quotes Readds quotes and apostrophes to HTML-encoded strings.
/proc/add_leading Adds 'char' ahead of 'text' until there are 'count' characters total.
/proc/add_trailing Adds 'char' behind 'text' until there are 'count' characters total.
/proc/sanitize_css_class_name Removes all non-alphanumerics from the text, keep in mind this can lead to id conflicts.
