/proc/area_is_occupied Checks if any living humans are in a given area.
/proc/hear Like view but bypasses luminosity check.
/proc/trange Alternative to range (ONLY processes turfs and thus less intensive).
/proc/circleviewturfs Is there even a diffrence between this proc and circlerangeturfs()?
/proc/recursive_content_check Will recursively loop through an atom's contents and check for mobs, then it will loop through every atom in that atom's contents. It will keep doing this until it checks every content possible. This will fix any problems with mobs, that are inside objects, being unable to hear people due to being in a box within a bag.
/proc/get_mobs_or_objects_in_view Returns a list of mobs and/or objects in range of R from source. Used in radio and say code.
/proc/get_mobs_and_objs_in_view_fast Uses dview to quickly return mobs and objects in view, then adds additional mobs or objects if they are in range 'smartly', based on their presence in lists of players or registered objects Type: 1-audio, 2-visual, 0-neither
/proc/flick_overlay_view Wrapper for flick_overlay, flicks to everyone who can see the target atom.
/proc/get_cardinal_step_away Returns the position of a step from start away from finish, in one of the cardinal directions.
/proc/get_active_candidates Will return a list of active candidates. It increases the buffer 5 times until it finds a candidate which is active within the buffer.
/proc/get_alien_candidates List of candidate KEYS to assume control of the new larva ~Carn Same as above but for alien candidates.
/proc/ScreenText List of candidate KEYS to assume control of the new larva ~Carn
/proc/getOPressureDifferential Gets the highest and lowest pressures from the tiles in cardinal directions around us, then checks the difference.
/proc/get_bbox_of_atoms Used for the multiz camera console stolen from virgo.
