/proc/prep_for_persist Prep for save: returns a preferences object if we're ready and allowed to save this mob.
/proc/resolve_excess_nutrition Resolve any surplus/deficit in nutrition's effet on weight all at once. Normally this would slowly apply during the round; once we get to the end we need to apply it all at once.
/proc/persist_nif_data Persist any NIF data that needs to be persisted. It's stored in a list to make it more malleable towards future shenanigans such as upgradable NIFs or different types or things of that nature, without invoking the need for a bunch of different save file variables.
